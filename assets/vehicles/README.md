# /assets/vehicles/

Coloca aquí las imágenes de los vehículos.

Nombres sugeridos según los autos del catálogo:
- corolla-levin.jpg   → Toyota Corolla Levin Sport
- byd-han.jpg         → BYD Han EV
- huracan.jpg         → Lamborghini Huracán Evo
- porsche-911.jpg     → Porsche 911 Turbo S
- jetour-t2.jpg       → Jetour T2
- g63.jpg             → Mercedes-Benz G 63 AMG
- nio-et7.jpg         → Nio ET7
- ferrari-roma.jpg    → Ferrari Roma
- lynkco-01.jpg       → Lynk & Co 01 Hybrid

Tamaño recomendado: 1200x800px, formato JPG, calidad 85.

Para usar una imagen local en lugar de Unsplash, 
reemplazá la URL en main.js (array CARS > img) o en el HTML directamente:
  'assets/vehicles/corolla-levin.jpg'
